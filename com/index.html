Component Object Model (COM) projects:


Compact

Compact is a class library for developing Visual C++ software components using the Component Object Model (COM). It provides a minimal set of mechanisms for implementing COM interfaces like IUnknown, IDispatch and IClassFactory.



Mapack for COM

Mapack is a COM component that provides classes and interfaces for matrix computations. supporting full, profile and symmetric matrix operations for the following numeric methods: Eigenproblem solver, Gauss method, Cholesky method, Gauss-Seidel, Conjugate-Gradient, Jacobi, Subspace interation.
