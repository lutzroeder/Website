Virtual Reality Aided Modeller

The Virtual Reality Aided Modeller (VRAM) was a research project at Bauhaus-Universität Weimar in 1998/99. The goal was to develop a virtual reality reasearch environment for 3D user interface design. More information can be found at the VRAM website.
