Imago

Project Imago developed the Orpheus track editor and Wave sound editor to create soundtracks for computer games and media presentations.

Tracks

Imago Orpheus

Imago Orpheus is/was a track based music editor and software synthesizer that allows editing of music tracks in a matrix-like editor.

Demo

Imago Wave

Imago Wave is/was a sample editor supporting RAW/WAV formats and effects liks echo, delay, chorus, reverb, vibrato, robotize and filter.
