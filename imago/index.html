Project Imago was developing software and soundtracks for computer games and media presentations. We developed the Orpheus track editor and the Wave sound editor and cooperated with game and media companies in creating soundtracks.

Imago Orpheus

Imago Orpheus is/was a track based music editor and software synthesizer that allows editing of music tracks in a matrix-like editor. Watch here.



Imago Wave

Imago Wave is/was a sample editor. Effects supported to modify the sound: echo, delay, chorus, reverb, vibrato, robotize, filter. Sample formats: RAW and WAV.
